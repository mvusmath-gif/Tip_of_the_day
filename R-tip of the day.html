What is aRtsy?

aRtsy is an r pacakge used to create generative art using mathematics, algorithms, randomness, and data-based patterns. This package can be used to:

  • Create abstract, mathematical art
  • Visualise data creatively
  • Experiment with algorithms and symmetry
  • Combine programming with design

aRtsy Example

library(aRtsy)

canvas_flow(
  colors = colorPalette("blossom"),
   background = "#fafafa",
  lines = 900,
  lwd = 0.05,
  iterations = 100,
  stepmax = 0.01,
  outline = c("none", "circle", "square"),
  polar = FALSE,
  angles = NULL
)

What is highcharter?

Highcharter is an R package that serves as a wrapper for the “Highcharts” JavaScript charting library. It creates interactive graphs and animated visualisations.

Highcharter also supports many different graph types like:

  • Line graphs
  • Scatterplots
  • Bar charts
  • Pie charts
  • Heat maps

Unlike normal graphs, highcharter has interactive elements that allow you to:

  • Hover over points
  • Zoom into graphs
  • Hide or show categories

Example 1: Function Explanation

  • It is very similar to ggplot and easy to use
  • Incrementally highlight additional lines
library(highcharter)
library(gapminder)
library(dplyr)

big4 <- gapminder |>
  filter(country %in% c("Niger","Swaziland", "Tanzania", "South Africa")) |>
  arrange(year)

# Creates empty highcharters object  
highchart() |>

# Inputs variables needed for graph 
  hc_add_series(data = big4,
                type = "line", 
                hcaes(x = year,
                      y = pop, 
                      group = country)) |>
  
  # Sets labels for axis
  hc_xAxis(title = list(text="Year")) |>
  hc_yAxis(title = list(text="Population")) |>
  hc_plotOptions(series = list(marker = list(symbol = "circle"))) |>
  hc_legend(align = "right", 
            verticalAlign = "top")
